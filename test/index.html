Now you can open your javascript console and run fetch('/proposal'). This will:
  1. Load an empty stub proposal and put it in the cache
  2. In the background, load the actual /proposal
    Meanwhile, is_loading(this.props) will return true, and a React component can use this to know it should display a loading indicator.
  3. Once the proposal has finished loading, the cache will automatically update and the global re_render() function will get called, which you can define to re-render your page.
